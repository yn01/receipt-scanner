# UI設計書 — 領収書スキャンWebアプリ（MVP）

## 1. デザイン方針

### 1.1 基本方針

- **モバイルファースト**: スマートフォンでの操作を最優先に設計し、タブレット・デスクトップへ拡張する
- **シンプル・直感的**: 最小限のステップでレシート撮影→保存を完了できるフローを重視
- **TailwindCSS**: ユーティリティファーストCSSで一貫したデザインを効率的に実装
- **タッチ最適化**: タップターゲットは最低44px、スワイプ等のジェスチャーは最小限に抑える

### 1.2 カラースキーム

| 用途 | 色 | TailwindCSS | 説明 |
|------|-----|-------------|------|
| Primary | `#2563EB` | `blue-600` | メインアクション、ブランドカラー |
| Primary Dark | `#1D4ED8` | `blue-700` | ホバー・アクティブ状態 |
| Secondary | `#F3F4F6` | `gray-100` | 背景、カード |
| Text Primary | `#111827` | `gray-900` | 本文テキスト |
| Text Secondary | `#6B7280` | `gray-500` | 補助テキスト、ラベル |
| Success | `#059669` | `emerald-600` | 保存完了、信頼度高 |
| Warning | `#D97706` | `amber-600` | 信頼度中、注意 |
| Error | `#DC2626` | `red-600` | エラー、削除 |
| Background | `#FFFFFF` | `white` | ページ背景 |
| Surface | `#F9FAFB` | `gray-50` | セクション背景 |

### 1.3 タイポグラフィ

| 要素 | サイズ | Weight | TailwindCSS |
|------|--------|--------|-------------|
| ページタイトル | 24px | Bold | `text-2xl font-bold` |
| セクション見出し | 18px | Semibold | `text-lg font-semibold` |
| 本文 | 16px | Normal | `text-base` |
| 補助テキスト | 14px | Normal | `text-sm text-gray-500` |
| キャプション | 12px | Normal | `text-xs text-gray-400` |

フォント: `font-sans`（TailwindCSS デフォルト — Inter, system-ui等）

### 1.4 スペーシング

- コンテンツ左右パディング: `px-4`（16px）
- カード間マージン: `space-y-3`（12px）
- セクション間: `space-y-6`（24px）
- ボタン内パディング: `px-4 py-3`（大）/ `px-3 py-2`（小）

## 4. コンポーネント設計

### 4.1 共通コンポーネント

#### Header

```typescript
type HeaderProps = {
  title: string;
  showBackButton?: boolean;
  onBack?: () => void;
  rightAction?: React.ReactNode;
};
```

- 固定ヘッダー（`sticky top-0`）
- 背景: `white`、下ボーダー: `border-b border-gray-200`
- 高さ: `h-14`

#### BottomNav

```typescript
type BottomNavProps = {
  activeTab: 'home' | 'scan';
};
```

- 固定フッター（`fixed bottom-0`）
- 2タブ: 一覧（ホーム）、スキャン
- アクティブ: `blue-600`、非アクティブ: `gray-400`
- 高さ: `h-16`（Safe Area考慮: `pb-safe`）

#### Button

```typescript
type ButtonProps = {
  variant: 'primary' | 'secondary' | 'danger' | 'ghost';
  size: 'sm' | 'md' | 'lg';
  fullWidth?: boolean;
  loading?: boolean;
  disabled?: boolean;
  children: React.ReactNode;
  onClick?: () => void;
};
```

| variant | 背景 | テキスト |
|---------|------|---------|
| primary | `bg-blue-600` | `text-white` |
| secondary | `bg-gray-200` | `text-gray-900` |
| danger | `bg-red-600` | `text-white` |
| ghost | `bg-transparent` | `text-blue-600` |

#### Card

```typescript
type CardProps = {
  children: React.ReactNode;
  onClick?: () => void;
  className?: string;
};
```

- スタイル: `bg-white rounded-lg shadow-sm border border-gray-200 p-4`

#### Modal

```typescript
type ModalProps = {
  isOpen: boolean;
  onClose: () => void;
  title: string;
  children: React.ReactNode;
  actions?: React.ReactNode;
};
```

- 背景オーバーレイ: `bg-black/50`
- モーダル本体: 下からスライドイン（モバイル）/ 中央表示（デスクトップ）

#### FAB (Floating Action Button)

```typescript
type FABProps = {
  onClick: () => void;
```

### 5.3 撮影フィードバック

| フィードバック | 実装方法 |

```typescript
const triggerShutterEffect = () => {
```
